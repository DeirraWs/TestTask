# Project

## Backend Settings:

1. Backend files are located without a directory.
2.Create a .env file with the following content:

   ```env
   BE_PORT=7000

   POSTGRES_HOST=localhost
   POSTGRES_PORT=5432
   POSTGRES_USERNAME=postgres
   POSTGRES_PASSWORD=root
   POSTGRES_DATABASE=test
 3. Run 
npm i 
npm run start:dev
